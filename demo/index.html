nanobar.js

Very very lightweight progress bars (~830 bytes gzipped).
No jQuery needed.

For iE9+ and the rest of the world


Example

var options = {
	bg: '#acf',

	// left target blank for global nanobar
	target: document.getElementById('myDivId'),

	// id for new nanobar
	id: 'mynano'
};

var nanobar = new Nanobar( options );

// move bar
nanobar.go( 30 ); // size bar 30%

// Finish progress bar
nanobar.go(100);
